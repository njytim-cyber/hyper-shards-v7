Please place your MP3 files in this directory with the following names:

1. load.mp3   - For the Load/Start screen and Game Over
2. shop.mp3   - For the Shop screen
3. wave.mp3   - For normal waves
4. boss.mp3   - For boss waves

The game will automatically load and play these files.
Note: You may need to click on the game once to allow audio to start playing due to browser autoplay policies.
